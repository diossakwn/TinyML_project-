# TinyML_project-
Ce projet d’instrumentation explore l’utilisation du TinyML (Machine Learning embarqué) pour réaliser deux systèmes intelligents capables de reconnaître de phénomènes physiques ou des objets en temps réel à l’aide d’une carte microcontrôleur arduino nano 33 BLE. Le projet est divisé en deux parties principales : la classification de vibrations et  de composants électroniques.
