La seconde partie du projet consiste à créer un système capable de reconnaître automatiquement différents composants électroniques (comme des LED, résistances ou potentiométre) à l’aide d’une caméra connectée à l’Arduino. 
Les images sont utilisées pour entraîner un modèle de vision embarquée via la plateforme Edge Impulse, qui permet de générer un modèle optimisé pour l'arduino nano 33 BLE sense. Ce modèle est ensuite intégré dans le programme Arduino afin de réaliser la classification d’images directement sur la carte. 
Les résultats de classification sont ensuite transmis à Node-RED, qui permet de traiter les données et d’afficher un tableau de bord interactif. Ce dashboard maintient un compteur du nombre de composants détectés pour chaque classe et permet de visualiser les résultats en temps réel.
