L’objectif de cette première partie est de développer un système capable de reconnaître différents types de vibrations à partir des données mesurées par le capteur inertiel (IMU) intégré à une Arduino Nano 33 BLE. Pour cela, des données de vibrations sont d’abord collectées via l’Arduino en utilisant l’accéléromètre et  le gyroscope. Ces données sont ensuite utilisées pour entraîner un modèle de machine learning permettant de distinguer 2 types de vibrations 
Une fois le modèle entraîné, il est converti en TensorFlow Lite afin d’être embarqué directement sur la carte Arduino. Le système peut alors analyser les vibrations en temps réel et classifier automatiquement le type de vibration détecté.
